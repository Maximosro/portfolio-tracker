Configurar Telegram

Para recibir notificaciones de alertas por Telegram, necesitas configurar un bot:

  1. Abre Telegram y busca @BotFather
  2. Envía /newbot y sigue las instrucciones → recibirás un Token
  3. Abre tu bot y envíale cualquier mensaje (ej: "hola")
  4. Abre en el navegador: https://api.telegram.org/bot{TOKEN}/getUpdates
  5. Busca "chat":{"id":XXXXXXX} → ese es tu Chat ID

Importar Operaciones

Instrucciones
  1. Prepara un archivo JSON con un array de operaciones (compras y ventas).
  2. Cada operación debe tener al menos: ticker, date, shares, price y type.
  3. Los campos name, yahooTicker, sector y color son opcionales pero recomendados (se usan para crear la posición).
  4. Si un ticker no existe en el portfolio, se creará automáticamente como nueva posición.
  5. Las posiciones se recalculan automáticamente a partir de las operaciones importadas (shares, precio medio, etc.).
  6. Pega el JSON abajo o sube un archivo .json.
🚨
¡Importante! Yahoo Ticker necesario para precios automáticos
Es absolutamente recomendable editar posteriormente las posiciones que no tengan el yahooTicker correcto para que los precios puedan actualizarse automáticamente. Sin este campo, tendrás que actualizar los precios manualmente.
Esquema y ejemplo del JSON
[
  {
    "ticker": "AAPL",           // ✅ Obligatorio — Ticker del activo (máx. 10 chars)
    "date": "2024-01-15",       // ✅ Obligatorio — Fecha en formato YYYY-MM-DD
    "shares": 10,               // ✅ Obligatorio — Número de acciones (> 0)
    "price": 150.50,            // ✅ Obligatorio — Precio por acción en € (> 0)
    "type": "BUY",              // ✅ Obligatorio — "BUY" (compra) o "SELL" (venta)
    "name": "Apple Inc.",       // ⭐ Opcional — Nombre del activo (para nuevas posiciones)
    "yahooTicker": "AAPL",      // ⭐ Opcional — Ticker de Yahoo Finance (para precios)
    "sector": "Tecnología",     // ⭐ Opcional — Sector o temática
    "color": "#5591c7"          // ⭐ Opcional — Color en hex para gráficos
  },
  {
    "ticker": "AAPL",
    "date": "2024-03-10",
    "shares": 5,
    "price": 165.00,
    "type": "BUY"
  },
  {
    "ticker": "AAPL",
    "date": "2024-06-20",
    "shares": 3,
    "price": 195.00,
    "type": "SELL"
  },
  {
    "ticker": "MSFT",
    "date": "2024-02-01",
    "shares": 8,
    "price": 380.25,
    "type": "BUY",
    "name": "Microsoft Corp.",
    "yahooTicker": "MSFT",
    "sector": "Tecnología",
    "color": "#6daa45"
  }
]
Campos
ticker — Identificador del activo (ej: AAPL)
date — Fecha YYYY-MM-DD (ej: 2024-01-15)
shares — Nº de acciones (ej: 10)
price — Precio por acción en € (ej: 150.50)
type — "BUY" o "SELL"
name — Nombre legible (opcional)
yahooTicker — Yahoo ticker para precios (opcional)
sector — Sector / temática (opcional)
color — Color hex para gráficos (opcional)
